## Simple backlight utility in C
Name says it all

### Usage
```bash
# get current backlight level
$ backlight

# set backlight
$ backlight 300

# increase or decrease backlight
$ backlight +30
$ backlight -30

# toggle backlight on or off
$ backlight on
$ backlight off
$ backlight toggle
```

### Installation
```bash
# to compile
$ make

# to install to /usr/bin
$ sudo make install
```
